template

A class-based HTML/CSS/JS responsive UI development framework for making webdoc.com apps and widgets.

Resources

repository
github.com/webdoc/template
documentation
webdoc.github.com/template

Template is essentially a library of classes that encapsulate layout techniques, some of them with javascript behaviours attached, intended for use as base classes in OOCSS-style development, but also useable as mixins.

Template gives you flexible columns with arbitrary content order, thumbs with hidden text, horizontal lists, popups that adapt to the size of their contents, drop-downs, tabs and slides, layers that fill the window, and more.

.slide

Slides are functionally identical to tabs – only one can be active at any one time. They differ in default style, where slides have the popular slide-in-from-the-side animation when they are activated. Slides also react to the special #prev and #next links.

Note that slides don't by default have a parent with overflow: hidden;. Usually you want one of those – but not always.

.wrap

The .wrap class is very basic. It is there to act as a position parent. It also has margin-left: auto; margin-rght: auto;, so that when it is given explicit width it centres itself horizontally.

.clear, .left and .right

Template doesn't use floats itself, but for convenience it does include the popular clear fix, and the classes left and right for floating content.
